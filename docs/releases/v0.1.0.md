# v0.1.0 Portfolio Release

PointCloud-GeoLab v0.1.0 is a portfolio release focused on reproducible point
cloud geometry demos, reviewable core implementations, and honest project
boundaries.

## Highlights

- CI runs on Python 3.10, 3.11, and 3.12.
- Coverage gate is enforced at 65% through `pytest-cov`.
- Stable public API exports are documented in `docs/api.md`.
- Portfolio pipeline generates a report, metrics JSON, figures, processed cloud,
  and transform JSON from deterministic demo data.
- Benchmark verifier checks CSV, JSON, Markdown, and PNG artifacts.
- Feature registration fallback is opt-in and diagnostic.
- GICP wording is explicit: this is GICP-style covariance-weighted ICP, not a
  full nonlinear GICP optimizer.

## Verification

```bash
python -m pip install -e ".[dev,vis,bench]"
make verify-core
make verify-portfolio
make verify-benchmarks
```

On systems without GNU Make, use the equivalent commands in
`docs/reviewer_checklist.md`.

## Known Limits

- Synthetic demos are smoke evidence, not broad real-data validation.
- Stanford Bunny, KITTI, and ModelNet are documented workflows requiring local
  datasets under `data/external/`.
- Optional Open3D, PyTorch, Plotly, SciPy, and scikit-learn paths are baselines
  or demos rather than core correctness evidence.
